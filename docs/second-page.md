### Маркированный список

- Первый пункт списка
- Второй пункт списка

- Третий пункт

1. Первый шаг
2. Второй шаг

4. Четвёртый шаг

## Блоки кода

### Python

```python
def hello_world():
    """Функция приветствия."""
    message = "Hello, World!"
    print(message)
    return message

if __name__ == "__main__":
    hello_world()
```

### JavaScript

```javascript
const greeting = (name) => {
    return `Hello, ${name}!`;
};

console.log(greeting('World'));
```

### Bash

```bash
#!/bin/bash
echo "Building project..."
mkdocs build
echo "Done!"
```
